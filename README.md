# Project-1---User-Input-Validation
Description

This is a beginner Python project that collects and validates user information, including first name, last name, age, and education level.
The program checks that names only contain letters, verifies that age is a valid number and meets a minimum age requirement, and converts education selections into readable values using a dictionary.

Purpose

I created this project to practice:
- while loops
- user input validation
- string methods (.isalpha(), .isdecimal(), .strip())
- dictionaries
- functions
- conditional logic
- code organization and readability
